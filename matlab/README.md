# National Flag of Nepal (MATLAB)

The most mathematical flag in the world.

Constitution of Nepal, Schedule 1, Article 8.

## Usage

```matlab
cd matlab
flagOfNepal
flagOfNepal(920)
flagOfNepal(920, 'color')
flagOfNepal(920, 'skeleton')
flagOfNepal(920, 'landmark')
```

| Mode | Description |
|------|-------------|
| `color` | Final coloured flag (default) |
| `skeleton` | Wireframe outline |
| `landmark` | Labels and imaginary construction lines |
